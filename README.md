# DLZ Studios Webpage  


### Dev setup  

#### npm  

* Setup workspace:  
`git clone https://github.com/lukasz26671/DLZStudiosWebpage.git`  
* Install packages:  
`npm install`  
* Open two powershell/cmd windows  
  - Start a server:  
`npm start`  
  - Start webpack bundler:  
`npm run react-dev`
